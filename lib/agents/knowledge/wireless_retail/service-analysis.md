# Service Analysis

## What is Service Analysis?
The Service Analysis page provides a downloadable PDF document with sales and service performance data. It's a static reference sheet, not a live dashboard.

## How do I use it?
1. Go to **Service Analysis** from the nav.
2. Tap the button to open or download the PDF.
3. On Android, the system share sheet opens so you can print, save, or share the document.
4. On iOS and web, the PDF opens in a new tab/window.

## Who can access it?
All retail roles can view the Service Analysis page.
