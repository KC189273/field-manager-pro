# Tasks

## How Tasks Work
- DMs and above can create tasks and assign them to employees
- Tasks have a title, description, due date, and optional photo requirement
- Employees receive push notifications when assigned a new task

## Completing a Task
1. Go to Tasks from the nav menu
2. Find your assigned task
3. If a photo is required, upload the photo
4. Tap "Complete"
5. Your DM is notified of completion

## Recurring Tasks
- DMs can set tasks to recur daily, weekly, or monthly
- Recurring tasks auto-create on their schedule

## Task Reminders
- Overdue tasks trigger automatic reminder notifications
- DMs can manually send a reminder for any open task
